# 概要
- 最新の津波観測に関する情報を取得し、コンソールに出力します。ある程度エラー対策しましたがまだエラーが起きる可能性があります。
- 自動取得機能はありません。処理完了後キーを押すと再取得します。

更新履歴はreleaseを確認してください。
